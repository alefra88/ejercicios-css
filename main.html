Algoritmo de CSS

Cascada

La cascada en CSS hace referencia a cómo se aplican las reglas CSS a un documento HTML. Las reglas CSS más específicas tienen prioridad sobre las reglas menos específicas y se aplican a los elementos en consecuencia.

Especifidad

La especificidad en CSS se refiere a cuán específicas son las reglas CSS en comparación con otras reglas CSS. Las reglas con una mayor especificidad tienen prioridad sobre las reglas con una menor especificidad.

Herencia

La herencia en CSS se refiere a cómo los estilos se pasan de un elemento padre a sus elementos hijos. Los elementos hijos heredan los estilos de sus elementos padres, a menos que se les apliquen estilos específicos que los sobrescriban.
